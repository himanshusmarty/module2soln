Types of css

InLine Style Sheets

Inline Style Sheets are included with HTML element i.e. they are placed inline with the element. To add inline CSS, we have to declare style attribute which can contain any CSS property. Syntax: ..….

lnline Style Sheets

Also called as Internal style sheets. Embedded Style Sheets are used to apply same appearance to all occurrence of a specific element. These are defined in head element by using the style element.

External Style Sheets

External Style Sheets are the separate .css files that contain the CSS rules. These files can be linked to any HTML documents using tag with rel attribute.
